## 1. 拆书逻辑 (rhythm_analyzer.py v3)

### 1.1 核心方法

**规则统计 (零 LLM 成本)** + **LLM-as-Judge 抽样验证** (~10% 章节)

### 1.2 22 维逐章指标

#### A. 基础指标 (规则计算)
| 指标 | 计算方法 | 字段名 |
|------|------|------|
| 字数 | `len(text.replace("\n","").replace(" ",""))` | `wc` |
| 段数 | `split("\n\n")` 统计 | `para_count` |
| 平均段长 | `wc / para_count` | `avg_para_len` |
| 对话比 | `对话字符总长 / wc` | `dialogue_ratio` |
| 感叹号密度 | `(叹号数 + 问号数) / wc × 100` | `excl_density` |

#### B. 爽点检测 (正则关键词 — 6 个子类型)

```
打脸 (PLEASURE_FACE_SLAP):
  打脸|嘲讽|看不起|小瞧|轻视|不屑|冷笑|嗤笑|凭什么|你也配|就这|
  惊呆|震惊|瞪大|倒吸|不可思议|怎么可能|不可能！|傻眼|目瞪口呆|鸦雀无声

突破 (PLEASURE_LEVEL_UP):
  突破|晋级|进阶|升级|渡劫|突破瓶颈|实力暴涨|修为大增|顿悟|
  觉醒|解锁|开启|新生|蜕变|脱胎换骨

碾压 (PLEASURE_CRUSH):
  碾压|秒杀|横扫|秒败|一击|一招|摧枯拉朽|不堪一击|
  螳臂当车|随手|轻易|不费吹灰|挥手之间

绝地反击 (PLEASURE_COMEBACK):
  绝地|绝境|反杀|逆转|翻盘|反败为胜|逆转乾坤|置之死地|
  破而后立|柳暗花明|峰回路转|起死回生|绝处逢生

扮猪吃虎 (PLEASURE_HIDDEN):
  扮猪|藏拙|隐藏实力|低调|显露|暴露|亮出|底牌|
  真正实力|终于出手|不再隐藏

通用爽感 (PLEASURE_GENERAL):
  哈哈|爽|痛快|舒服|牛逼|厉害|很强|极强|无敌|逆天|
  恐怖如斯|骇然|惊呼|赞叹|佩服|崇拜|仰望|敬畏
```

**爽点密度**: `total_pleasure_matches / wc × 100` → `pos_density`
**主导爽点子类型**: `argmax(各子类型计数)` → `dominant_sub`

#### C. 冲突检测
```
冲突词正则:
  你敢|休想|去死|找死|不可能！|我不信|凭什么|
  战斗|杀|轰|爆|碎|血|伤|敌|战|斗|
  杀意|杀气|怒吼|暴怒|杀机|出手|进攻|攻击|反击|
  一刀|一剑|一拳|一掌|一枪|武器|兵器|法宝|神通|
  拼命|拼死|搏命|殊死|血战|激战|苦战|酣战|大战|
  对决|决斗|生死|致命|致命一击|决一死战|不死不休
```

**冲突密度**: `conflict_matches / wc × 100` → `conflict_density`
**冲突等级**: `>2.5→high | >1.0→medium | >0.3→low | else→none`

#### D. 章末钩子检测 (4 类 — 马良写作分类法)

检查章末 500 字符:

```
悬念式: 竟然|居然|突然出现|不可能|怎么可能  (章末100字)
反转式: 长句后跟短句的句式突变      (章末200字)
情绪炸弹: 从[来没]|你[就从没]|再也[不没]+短句  (章末100字)
信息投放: 翻开|打开|看到|发现+短句  (章末100字)
```

**钩子密度**: `全章 cliffhanger 词匹配数 / (wc/1000)` → `hook_density`
**钩子类型**: 优先顺序 `悬念式 > 情绪炸弹 > 反转式 > 信息投放 > none`

#### E. 文笔可读性 (AlphaReadability Chinese 简化版)

```python
# 句长复杂度
sentences = re.split(r'[。！？!?]', body)
avg_sentence_len = mean([len(s.strip()) for s in sentences if s.strip()])

# 词汇多样性
unique_chars = len(set(pure_text))
vocab_diversity = unique_chars / max(len(pure_text), 1)

# 综合可读性 0-1 (越高越复杂/文学性)
readability = min(1.0,
    (avg_sentence_len / 80) * 0.5 +
    (1 - vocab_diversity * 10) * 0.3 +
    (abs(avg_sentence_len - 35) / 50) * 0.2
)
```

#### F. 爽点强度 (校准公式)

```python
pleasure_intensity = max(0, min(10,
    pos_density * 1.5 +          # 爽点关键词密度
    excl_density * 1.0 +         # 感叹号=兴奋度
    dialogue_ratio * 8 +         # 对话密集=角色互动
    hook_density * 0.5 -         # 悬念增加张力
    neg_density * 0.3            # 负面情绪微降爽感
))

# 爽点类型 (阈值切割)
>=6 → "climax" (高潮)
>=4 → "major"  (重要爽点)
>=2 → "minor"  (微爽点)
<2  → "none"
```

#### G. 三级爽点递进 (马良写作 3-ch 滑动窗口)

```python
window = 前后各1章 (共3章)
avg_intensity = mean([窗口内所有 intensity])
has_climax = any(窗口内有"climax")

if has_climax and avg_intensity >= 6 → "large"  (大爽点)
elif has_major and avg_intensity >= 4 and slap_total >= 3 → "medium"
elif slap_total >= 1 or intensity >= 2 → "small"
else → "none"
```

#### H. 情绪 & 节奏

| 情绪分类 | 条件 |
|------|------|
| 爽快 | `pos_density > neg_density × 2` |
| 紧张 | `conflict_density > 2 AND neg_density > pos_density` |
| 悲壮 | `conflict_density > 1.5` |
| 悬疑 | `comeback_count > slap_count` |
| 日常 | `dialogue_ratio > 0.35` |
| 日常 | else |

| 节奏 | 条件 |
|------|------|
| fast | `avg_para_len < 45 OR (avg_para_len < 80 AND excl_density > 0.5)` |
| slow | `avg_para_len > 250` |
| medium | else |

#### I. 章间变异性 (DTW 简化 — Applied Sciences 论文启发)

```python
ch_variability = mean([
    |wc_diff| / max(prev_wc, 1),
    |dialogue_ratio_diff| / max(prev_dialogue, 0.001),
    |pos_density_diff| / max(prev_pos, 0.001),

```

#### J. LLM-as-Judge 验证 (WebNovelBench 方法)

- 每本书约 10% 章节抽样 (步长=`total//10`, 上限 15 章)
- 必含: 首章、末章、最高爽点章、最高冲突章

## 2. 商业评分逻辑 (genre_synthesizer.py v6)

### 2.1 火书池构建

从 5 本末世火书 CSV 中按章节位置提取指标，计算 P25/P50/P75 基准:

```python
# 每本书提取:
前3章 hook_density 均值, conflict_density 均值, pleasure_intensity 均值
前30章 打脸频率 (slap_count/30)
全书 Shannon 爽点多样性指数
全书 反转式钩子率, 悬念式钩子率, large级爽点率

# 池统计 (5 本书的指标向量 → 排序 → P25/P50/P75)
firebook_pool = {
    "hook_density":  {p25, p50, p75},
    "conflict":      {p25, p50, p75},
    "intensity":     {p25, p50, p75},
    "slap_rate":     {p25, p50, p75},
    "diversity":     {p25, p50, p75},
    "reversal_rate": {p50},
    "suspense_rate": {p50},
    "large_rate":    {p50},
    "n_books": 5
}
```

### 2.2 百分位制评分函数

```python
def percentile_score(value, pool, metric):
    """P50 = 50分, P25 = 25分, P75 = 75分. 线性映射."""
    p25, p50, p75 = pool[metric]["p25"], pool[metric]["p50"], pool[metric]["p75"]
    if p75 == p25:
        return 50
    pct_rank = (value - p25) / (p75 - p25) * 50 + 25
    return round(clamp(pct_rank, 0, 100))
```

### 2.3 9 维度评分卡

| 维度 | 计算 | 数据源 | 评分方式 |
|------|------|------|:---:|
| **前3章钩子** | `mean(hook_density[0:3])` 前3章 | 火书池 P25/P50/P75 | 百分位 |
| **前3章冲突** | `mean(conflict_density[0:3])` 前3章 | 火书池 P25/P50/P75 | 百分位 |
| **首章爽点** | `mean(pleasure_intensity[0:3])` 前3章 | 火书池 P25/P50/P75 | 百分位 |
| **零钩子连续** | 前30章最大连续零钩子章数 | 硬编码 | ≤2章=100分, 每+1章-30分 |
| **打脸频率** | 前30章 slap_count/30 | 火书池 + 最优区间 | 0.25-0.5次/章=90分, 否则百分位 |

### 2.4 综合加权 & 评级 (番茄平台真实权重)

  >=50 → "✅ 签约可期"

## 4. 评审问题清单

### 关于拆书逻辑
