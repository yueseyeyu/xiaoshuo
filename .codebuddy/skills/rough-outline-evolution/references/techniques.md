## 刘丙润技法

### 主线一句话法
主角（姓名）想要（具体目标），但（核心阻碍），如果失败（具体代价）。

### 前500字检查清单
- □ 出现主角名字/代称？
- □ 出现至少1个具体动作？
- □ 埋入1个"读者想知道答案"的疑问？
- □ 出现1个感官细节？
- □ 读者感受到"危险/好奇/爽感"中≥1种？

### 期待感锚点四类
- 信息差：读者知道但主角不知道（或反之）
- 时间锁：必须在XX时间内完成XX
- 代价：失败的具体后果，越具体越紧张
- 成长：让读者预判主角即将变强/反杀

### 伏笔三链
预埋点 → 触发点 → 回收点（缺一则逾期失效）

### 转场三模板
- A-时间跳转："那是XX小时前的事。现在，XX。"
- B-空间跳转："XX并不知道，在XX，XX正在发生。"
- C-视角切换："XX看着监控画面里的主角，嘴角上扬。"

### 清冷角色对话原则 (月的专属模板)
1. 少即是多：用动作代替台词
2. 潜台词：不直接表达真实想法
3. 不说废话：砍掉语气词和过渡词

### 书名简介标准
- 书名：13-15字，震惊/猎奇/趣味/噱头感
- 简介：3句话，每句一个转折

## 番茄平台标准

### 读者弃书5大死因
1. 节奏拖沓/注水
2. 人设崩塌
3. 强行降智
4. 毒点
5. 期待感枯竭
- 弃书高发区：1-3章 / 10-20章 / 收费前后 / 换地图 / 新卷

## v5.7创作技能 (2026.6)

### 逆向操控法
1. 三根高压线：每条世界规则直接催生冲突
2. 关系方程式：角色A对B的欲望+阻碍+后果
3. 反推法章纲：从卷高潮反推事件链
4. 落差公式：情绪=期望-现实
