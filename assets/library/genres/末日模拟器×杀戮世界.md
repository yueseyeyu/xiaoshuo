# 《末世模拟器：开局看到自己死在第七天》

> 书名 14 字 · 震惊体 · 番茄黄金字数

## 题材组合
- 主赛道: 科幻末世
- 金手指: 模拟器（消耗"气"试错未来）
- 风格: 生存文（非无敌文）

## 世界观速览
- **灾难机制**: 侵蚀气体来自"侵蚀源"（植物/动物/物体），每个侵蚀源有固定侵蚀范围 → 植物→动物→人类层层进化
- **进化二分**: 进化伴随杀戮嗜血侵蚀 → 扛住了=进化+保留意识 / 扛不住=失控（嗜血怪物，非妖）
- **异能来源**: ①自身天赋 ②身体/知识技能转化 ③进化源（摧毁侵蚀源掉落的核心材料）
- **代价**: 理智持续被侵蚀，越强越接近失控临界点
- **对抗侵蚀**: 靠自身意志力/本能强度。模拟器可预知侵蚀高峰提前规避。无外挂药物
- **侵蚀源机制**: 
  - 可摧毁 → 获得进化材料/异能觉醒关键
  - 新源不断出现，旧源持续成长（浓度到阈值→进化）
  - 源的本质: 杀戮世界的"定位器"——清源=生存，但也会让剩余源的定位校准加速
- **爆发节奏**: 渐进式——气体浓度渐增，每个人/生物的"失控临界点"不同
- **终极冲突**: 杀戮世界入侵——气体就是侵略第一步
- **模拟器**: 无消耗代价，但①时间限制 ②层级限制（低级时模拟不到高位存在，即使遇见也会被杀）
- **权力来源**: 本能驾驭 → 进化等级越高越强，但侵蚀同步增强 → 不进化=死，进化=接近失控
- **矛盾体系**: 
  - 终极矛盾(A): 人类 vs 杀戮世界入侵者（种族对抗）
  - 前期矛盾(B): 进化者 vs 失控者（范围最广）
  - 前期矛盾(C): 人性 vs 进化（每个人内心的拉锯）
- **妖**: 动植物进化诞生灵智的总称。核心：柳树妖（清冷如月，异能月光）。与人类互不信任但同盟
- **场景**: 
  - 核心: 异化仪式(B) + 模拟者困境(C) → 日常被侵蚀+全知者负罪
  - 活动区域: 城市小区（初期）→ 郊区基地（模拟后搬家，囤货，觉醒异能）
- **倒计时**: 主角通过模拟器提前7天知晓末世降临
