1. pip install novel-downloader
2. 逐本下载TXT → data/novels/
3. python scripts/check_encoding.py
4. python scripts/convert_encoding.py
5. 更新 novel_index.json
6. rhythm_analyzer → llm_batch_score → genre_synthesizer
