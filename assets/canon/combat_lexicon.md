## 五、场景识别规则

用于拆书管线自动识别打斗场景：

```
检测到连续出现 ≥3 个"招式动作"词 → 标记为【打斗场景】
检测到"力量气势+光影特效+碰撞冲击"组合 → 标记为【高描写密度战斗】
统计每章打斗场景的词汇丰富度（去重后/总词数）→ 作为评分维度
```
