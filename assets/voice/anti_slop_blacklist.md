# AI 指纹黑名单

> 动态更新 · 用于 S4 检测（AI 指纹词/句法重复/风格漂移）

## 一、AI 高频指纹词

以下词汇在 AI 生成文本中高频出现，人类作者较少使用：

### 1.1 转折/递进类
- 然而、不过、尽管如此、与此同时、在此基础上
- 值得注意的是、不可否认、显而易见、毋庸置疑
- 换言之、简而言之、总而言之、综上所述

### 1.2 修饰/程度类
- 格外、尤为、颇为、甚为、极其
- 无疑、确实、显然、仿佛、似乎
- 一定程度、某种程度上、在某种意义上

### 1.3 情感/反应类
- 不禁、不由得、忍不住、情不自禁
- 微微一笑、嘴角上扬、眼中闪过
- 心中暗想、暗自庆幸、暗自点头

### 1.4 网文 AI 套路词
- 宛如洪荒猛兽、犹如潮水般、仿佛要吞噬一切
- 恐怖如斯、令人窒息、不寒而栗
- 毫无疑问、不言而喻、心知肚明
- 顿时、刹那间、一瞬间、霎时间（过度使用时）

## 二、句法重复模式

### 2.1 机械三段式
```
[描写环境] + [角色动作] + [内心独白]
```
AI 倾向每段都按此结构写，人类作者会有变化。

### 2.2 对话标签重复
```
"..."他说。
"..."她说。
"..."他回答道。
```
连续 3+ 段使用相同对话标签结构。

### 2.3 段落开头重复
连续 3+ 段以相同词性开头（如连续"他..."、"她..."）。

### 2.4 转折堆叠
单章使用 5+ 次"然而""不过""但是"等转折词。

## 三、风格漂移信号

| 信号 | 判定 | 阈值 |
|------|------|------|
| 句长突变 | 当前章平均句长与基线偏差 > 50% | deviation > 0.5 |
| 对话占比突变 | 当前章对话占比与基线偏差 > 20% | deviation > 0.2 |
| 词汇丰富度下降 | 当前章去重词数/总词数 < 基线 × 0.7 | ratio < 0.7 |
| 描写密度突变 | 当前章描写句占比与基线偏差 > 25% | deviation > 0.25 |

## 五、更新规则

1. 每次发现新的 AI 套路词，追加到对应分类
2. 每月从生成内容中统计高频词，与人类精品书对比
3. 人类作者常用但 AI 不用的词不需要加入（如方言、俚语）
4. 只加入"AI 高频但人类低频"的词
