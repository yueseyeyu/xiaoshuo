# 番茄小说 AI 辅助创作系统 —— v7.12 前端设计系统 v5

> 目标：从「通用管理后台」转向「专业创作者工作站」
> 视觉方向：克制深色、高信息密度、低卡片依赖、强内容层级
> 核心参考：Linear / Obsidian / 专业创作工具，减少 SaaS 仪表盘的卡片堆砌感

---

## 一、设计原则

1. **内容优先**：界面服务于小说创作与分析，装饰元素让位于信息本身。
2. **少卡片、多布局**：能用留白、分割线、列排版的区域，不用玻璃卡片包裹。
3. **一套主题变量，六套配色**：每套主题独立校准背景、强调色、文字、发光，避免简单色相旋转。
4. **实时反馈**：硬件、任务、设计建议等状态即时可见。
5. **AI 可介入骨架设计**：粗纲、细纲、世界观、角色、势力等设计模块明确提供 AI 优化建议与修改入口。


## 五、组件规范

### KPI 卡片

- 使用 `.kpi-row` 横向排列，桌面端最多 4-5 列。
- 数字大、标签小，避免多余装饰。
- 书库 KPI 中「总书数」需随题材过滤实时变化。

### 管线（Pipeline）

- 横向阶段条，每个阶段包含图标、状态点、名称。
- 阶段最大宽度 160px，避免宽屏拉伸变形。
- 当前运行阶段使用脉冲动画与强调色。

### 侧边卡片

- 用于聚集次要信息（系统健康、最近任务、统计）。
- 标题 + 内容， hover 仅切换背景与边框色，无上浮阴影。
- 避免单张卡片过高导致空白。

### 抽屉 / 模态

- **报告详情**：右侧滑出抽屉，宽 420px，用于编辑核心洞察与详细分析。
- **设计编辑**：居中大窗模态，宽 1000px，类似写作页沉浸体验。
- 所有抽屉/模态带遮罩，点击遮罩或关闭按钮可关闭。

### 表单字段

- label 在上，input/textarea 在下。
- 深色背景、浅色边框，聚焦时边框变为强调色。
- textarea 支持纵向拉伸。

## 六、页面设计说明

### 工作台（Dashboard）

- 桌面端采用 `1fr 360px` 两栏：左侧主信息，右侧辅助信息。
- Hero 区：左书名/题材/操作，右 3 个关键统计卡片。
- KPI 区：4 个横向紧凑卡片。
- 管线区：横向展示末世管线 10 个阶段。
- 最近动态：填充主区域，避免底部空白。

### 书库（Library）

- 顶部题材标签横向滚动，点击过滤书籍。
- KPI：总书数、已入库、待处理、题材数（全局）。
- 总书数 / 已入库随当前题材实时更新。
- 书籍卡片展示封面、书名、作者、字数、题材标签。

### 拆书（Disassembly）

- 顶部 KPI：总任务、进行中、已完成、失败。
- 拆书管线可视化：文本导入 → 章节拆分 → 节奏分析 → 技法提炼。
- 任务卡片网格展示各任务进度。
- 右侧边栏：待拆书队列、最近完成、拆书统计。

### 报告（Reports）

- 顶部过滤标签：全部 / 写作指导 / 市场分析 / 人物分析 / 节奏拆解 / 技法提炼。
- 主区域 2 列报告卡片，显示标题、摘要、标签、元信息。
- 右侧边栏：报告统计、最近生成、待处理洞察。
- 点击卡片打开右侧详情抽屉，可编辑并获取 AI 建议。

### 设计（Design）

- 左侧垂直标签：粗纲 / 细纲 / 世界观 / 角色 / 势力。
- 各标签下使用卡片网格展示内容。
- 点击整卡进入编辑（不再使用编辑按钮）。
- 编辑界面为 1000px 居中大窗模态，支持 AI 建议与保存。

### 写作（Writing）

- 普通模式：顶部工具栏 + 标题输入 + 正文 textarea + 右侧辅助面板。
- 专注模式：Fullscreen API 真全屏，隐藏顶部/侧边/chrome，工具栏 sticky 置顶。
- 专注工具栏：退出专注 / 去AI味 / AI对比 / 字数 / 保存草稿。
- 去AI味：底部提示条显示检查项。
- AI对比：右侧滑出专用面板，窄屏自适应宽度。

### 硬件（Hardware）

- 仪表盘网格展示 GPU/VRAM/CPU/RAM 使用率与温度。
- 下方实时日志面板。
- 卡片最大宽度 220px，避免宽屏过度拉伸。

### 设置（Settings）

- 桌面端 3 列卡片网格。
- 每张卡片一个设置主题：主题、模型、写作偏好、快捷键、数据、账户。

## 七、响应式策略

| 断点 | 行为 |
|------|------|
| <= 768px | 侧边栏变抽屉；各页面转为单列；KPI 单列；报告卡片单列 |
| 769px - 1200px | 侧边栏 icon-only；dashboard/report 转单列 |
| >= 1201px | 完整侧边栏；dashboard 双栏；report 双栏；settings 3 列 |
| 1920px 全屏 | 所有内容按创作工作站密度排布，重点消除大面积空白 |

## 八、动效规范

- 过渡时长统一 0.15s - 0.25s，使用 ease。
- 卡片 hover：背景变为 `--surface-hover`、边框变为 `--border-hover`，无上浮/无营销化阴影。
- 主按钮 hover：使用 `filter: brightness(1.08)`，无上浮。
- 运行中状态点使用脉冲动画。
- 抽屉/模态/下拉菜单使用 transform 滑入/缩放，避免重排。
